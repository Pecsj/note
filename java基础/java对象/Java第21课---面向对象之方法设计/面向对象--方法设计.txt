	面向过程  面向对象---编程思想的区别
	类和对象
	类-----抽象笼统的概念(人类)
		是用来描述很多对象  具有相同的特征(属性)和行为(方法)
	对象---类中衍生出来的一个具体的个体(人)
		应该与自己的类具有相同的特征和行为
		
	1.需要自己(其他人String Scanner Math)描述一个类Person
	2.利用属性或方法 描述类的具体形态
	3.创建一个当前类的对象  对象.调用属性  赋值/取值    调用方法 执行一次

	可以理解为参数是做事情之前  必须提供的条件
