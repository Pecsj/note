# 数据库语言分类
1. DDL(Date Definition Language)数据库定义语言
    * 用来定义数据库对象：数据库，表，列
    * 关键字：creat,drop,alter
2. DML(Date Manipulation Language)数据操作语言
    * 用来对数据库中表的数据进行增删改
    * 关键字：insert delete update
3. DQL(Date Query Language)数据查询语言
    * 查询数据库中表的记录
    * select where
4. DCL(Date Control Language)数据控制语言
    * 定义数据库的访问权限和安全级别
    * grant revoke
